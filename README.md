# 🗒️ My Notes App

A simple notes application built using Expo React Native for learning and practice.

---

## 🚀 Features
- Create notes
- Edit notes
- Delete notes
- Simple and clean UI

---

## 🛠️ Tech Stack
- React Native (Expo)
- JavaScript / TypeScript (jo bhi use kiya ho)

---

1. Install dependencies

   ```bash
   npm install
   ```
   
2. Start the app

   ```bash
   npx expo start
   ```
